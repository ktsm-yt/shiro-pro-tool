# 城プロ編成ツール

城プロのキャラクター管理と編成作成のためのWebアプリケーション

## 機能

- **キャラクター管理**: キャラクターの追加、編集、削除
- **編成作成**: 最大8人の編成を作成
- **バフ/デバフ可視化**: 編成のバフ/デバフ強度を棒グラフで表示
- **編成比較**: 複数の編成を比較
- **データ保存**: ブラウザのLocalStorageに自動保存
- **データエクスポート/インポート**: JSONファイルでデータをバックアップ・共有

## データ管理

### 保存場所
- キャラクターデータ: ブラウザのLocalStorage (`shiroProCharacters`)
- 編成データ: ブラウザのLocalStorage (`shiroProFormations`)

### データのエクスポート/インポート
キャラクター管理タブの「データ管理」セクションから以下の操作が可能です：

- **キャラクターデータをエクスポート**: キャラクターデータのみをJSON形式でダウンロード
- **編成データをエクスポート**: 編成データのみをJSON形式でダウンロード
- **全データをエクスポート**: キャラクターと編成の両方をまとめてダウンロード
- **インポート**: エクスポートしたJSONファイルを読み込み（既存データに追加/上書き）

### データフォーマット
`data/` フォルダにサンプルJSONファイルがあります。

#### キャラクターデータの構造
```json
[
  {
    "id": 1,
    "name": "キャラ名",
    "period": "期間限定の場合（嫁など）",
    "weapon": "武器種",
    "weaponRange": "遠 or 近",
    "weaponType": "物 or 術",
    "placement": "遠 or 近 or 遠近",
    "attributes": ["水", "平"],
    "skills": ["特技の配列"],
    "formationSkills": ["編成特技の配列"],
    "strategies": ["計略の配列"]
  }
]
```

## 使い方

1. `index.html` をブラウザで開く
2. キャラクターを追加
3. 編成を作成
4. データは自動的にブラウザに保存されます

## 開発

このツールは単一のHTMLファイルで動作します。
